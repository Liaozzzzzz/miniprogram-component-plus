# select-text

可选文本组件。

## 属性列表

| 属性   | 类型    | 默认值 | 必填 | 说明         |
| ------ | ------- | ------ | ---- | ------------ |
| space  | String  |        | 否   | 显示连续空格 |
| decode | Boolean | false  | 否   | 是否解码     |

#### space 的合法值

| 属性 | 类型                   |
| ---- | ---------------------- |
| ensp | 中文字符空格一半大小   |
| emsp | 中文字符空格大小       |
| nbsp | 根据字体设置的空格大小 |
